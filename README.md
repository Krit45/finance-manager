# 🚀 FinanceFlow — AI-Powered Personal Finance Tracker

> 💡 Take control of your money with real-time insights, smart budgeting, and a beautifully designed dashboard.

---

## 🌟 Overview

**FinanceFlow** is a modern, scalable personal finance management web app designed to help users track income, monitor expenses, and stay within budget effortlessly.

Built using cutting-edge technologies like **React, Firebase, and Vite**, the app delivers a seamless experience with real-time data syncing and powerful analytics.

---

## ✨ Features

### 💰 Smart Expense Tracking

* Add, edit, and delete transactions
* Categorize income & expenses
* Recurring transaction support

### 📊 Real-Time Analytics

* Interactive charts using Recharts
* Visual breakdown of spending habits
* Monthly budget insights

### 🔐 Secure & Scalable Backend

* Firebase Authentication & Firestore
* User-specific data isolation
* Real-time database updates

### 🎨 Modern UI/UX

* Clean and responsive design
* Dark & Light mode support
* Smooth animations (Motion)

### 🤖 AI-Ready Architecture

* Gemini API integration support
* Designed for future AI budgeting suggestions

## ⚙️ Environment Setup

Create a `.env` file and add:

```env
GEMINI_API_KEY=your_api_key_here
```

Also configure Firebase using:

```json
{
  "projectId": "finance-92fe1"
}
```

(Your Firebase config is already set up )

---

## 🚀 Run Locally

```bash
npm install
npm run dev
```

(Default runs on port 3000 )
